# TDEE Calculator 🔥

Precision TDEE (Total Daily Energy Expenditure) Calculator — เครื่องคำนวณแคลอรี่ที่ร่างกายใช้ต่อวัน

👉 **[https://YOUR_USERNAME.github.io/tdee-calculator/](https://YOUR_USERNAME.github.io/tdee-calculator/)**

## Features

- **4 สูตร BMR มาตรฐานวิทยาศาสตร์**: Mifflin-St Jeor, Harris-Benedict (Revised), Katch-McArdle, Cunningham
- **5 ระดับกิจกรรม**: จากนั่งทำงานถึงนักกีฬา
- **TDEE แยกตามประเภทวัน**: วันพัก, วันเวท, วันคาร์ดิโอ, วันเวท+คาร์ดิโอ (Calorie Cycling)
- **ผลลัพธ์ครบถ้วน**: TDEE, BMR, แคลอรี่ลด/รักษา/เพิ่มน้ำหนัก, สัดส่วนมาโครแนะนำ
- **เปรียบเทียบ 4 สูตร** พร้อมรายละเอียดสมการ
- **2-Page Flow**: หน้ากรอกข้อมูล → หน้าผลลัพธ์
- **Responsive** รองรับทุกขนาดหน้าจอ
